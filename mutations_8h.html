Create mutations to delete a range of cells from a column.

The following functions create a mutation that deletes all the cells in the given column family and column within the given timestamp range.

The ending timestamp is exclusive, while the beginning timestamp is inclusive. Please notice that [a,a) is an invalid range unless a==0.
