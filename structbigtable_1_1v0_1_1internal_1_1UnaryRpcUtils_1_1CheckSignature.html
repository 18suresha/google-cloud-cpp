Detailed Description

template<typename ClientType>
template<typename T>
struct bigtable::v0::internal::UnaryRpcUtils< ClientType >::CheckSignature< T >

Determine if T is a pointer to member function with the expected signature for MakeCall().

This is the generic case, where the type does not match the expected signature. The class derives from std::false_type, so CheckSignature<T>::value is false.
