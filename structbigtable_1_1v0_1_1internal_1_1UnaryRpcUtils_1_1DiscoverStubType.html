Detailed Description

template<typename ClientType>
struct bigtable::v0::internal::UnaryRpcUtils< ClientType >::DiscoverStubType

Extract the StubType from the type returned by ClientType::Stub().

This is a meta-function to convert the std::shared_ptr<T> type returned by ClientType::Stub into a plain T. We follow the STL convention of using type for the returned type of this metafunction.
