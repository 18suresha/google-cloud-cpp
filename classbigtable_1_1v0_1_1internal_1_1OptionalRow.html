Detailed Description

A poor's man version of std::optional<Row>.

This project needs to support C++11 and C++14, so std::optional<> is not available. We cannot use Abseil either, see #232 for the reasons. So we implement a very minimal "OptionalRow" class that documents the intent and we will remove it when possible.
