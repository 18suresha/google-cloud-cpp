Detailed Description

template<bool compile_time_enabled>
class google::cloud::v0::Logger< compile_time_enabled >

Define the class to capture a log message.

Template Parameters
compile_time_enabledthis represents whether the severity has been disabled at compile-time. The class is specialized for false to minimize the run-time impact of (and, in practice, completely elide) disabled logs.
