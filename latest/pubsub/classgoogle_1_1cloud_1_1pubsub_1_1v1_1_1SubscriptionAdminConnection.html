Detailed Description

A connection to Cloud Pub/Sub for subscriber operations.

This interface defines pure-virtual methods for each of the user-facing overload sets in SubscriberClient. That is, all of SubscriberClient's CreateSubscription() overloads will forward to the one pure-virtual CreateSubscription() method declared in this interface, and similar for SubscriberClient's other methods. This allows users to inject custom behavior (e.g., with a Google Mock object) in a SubscriberClient object for use in their own tests.

To create a concrete instance that connects you to the real Cloud Pub/Sub service, see MakeSubscriptionAdminConnection().
