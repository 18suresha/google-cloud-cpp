Detailed Description

Defines the interface to acknowledge and reject messages.

This interface allows applications to acknowledge and reject messages in a provided to the Cloud Pub/Sub C++ client library. Note that this class is move-able, to support applications that process messages asynchronously. However, this class is not copy-able, because messages can only be acknowledged or rejected exactly once.
