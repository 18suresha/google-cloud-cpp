38 /**
39  * Connects to Cloud Bigtable's table administration APIs.
40  *
41  * This class is used by the Cloud Bigtable wrappers to access Cloud Bigtable.
42  * Multiple `bigtable::TableAdmin` objects may share a connection via a
43  * single `AdminClient` object. The `AdminClient` object is configured at
44  * construction time, this configuration includes the credentials, access
45  * endpoints, default timeouts, and other gRPC configuration options. This is an
46  * interface class because it is also used as a dependency injection point in
47  * some of the tests.
48  *
49  * @par Cost
50  * Applications should avoid unnecessarily creating new objects of type
51  * `AdminClient`. Creating a new object of this type typically requires
52  * connecting to the Cloud Bigtable servers, and performing the authentication
53  * workflows with Google Cloud Platform. These operations can take many
54  * milliseconds, therefore applications should try to reuse the same
55  * `AdminClient` instances when possible.
56  */
