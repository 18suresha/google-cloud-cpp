37 /**
38  * Connects to Cloud Bigtable's table administration APIs.
39  *
40  * This class is used by the Cloud Bigtable wrappers to access Cloud Bigtable.
41  * Multiple `bigtable::TableAdmin` objects may share a connection via a
42  * single `AdminClient` object. The `AdminClient` object is configured at
43  * construction time, this configuration includes the credentials, access
44  * endpoints, default timeouts, and other gRPC configuration options. This is an
45  * interface class because it is also used as a dependency injection point in
46  * some of the tests.
47  *
48  * @par Cost
49  * Applications should avoid unnecessarily creating new objects of type
50  * `AdminClient`. Creating a new object of this type typically requires
51  * connecting to the Cloud Bigtable servers, and performing the authentication
52  * workflows with Google Cloud Platform. These operations can take many
53  * milliseconds, therefore applications should try to reuse the same
54  * `AdminClient` instances when possible.
55  */
