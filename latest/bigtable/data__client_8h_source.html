45 /**
46  * Connects to Cloud Bigtable's data manipulation APIs.
47  *
48  * This class is used by the Cloud Bigtable wrappers to access Cloud Bigtable.
49  * Multiple `bigtable::Table` objects may share a connection via a single
50  * `DataClient` object. The `DataClient` object is configured at construction
51  * time, this configuration includes the credentials, access endpoints, default
52  * timeouts, and other gRPC configuration options. This is an interface class
53  * because it is also used as a dependency injection point in some of the tests.
54  *
55  * @par Cost
56  * Applications should avoid unnecessarily creating new objects of type
57  * `DataClient`. Creating a new object of this type typically requires
58  * connecting to the Cloud Bigtable servers, and performing the authentication
59  * workflows with Google Cloud Platform. These operations can take many
60  * milliseconds, therefore applications should try to reuse the same
61  * `DataClient` instances when possible.
