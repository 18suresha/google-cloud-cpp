Detailed Description

Mock an instance of storage::internal::CurlRequest.

The mocking code is a bit strange. The class under test creates a concrete object, mostly because it seemed overly complex to have a factory and/or pass the object as a pointer. But mock classes do not play well with copy or move constructors. Sigh. The "solution" is to create a concrete class that delegate all calls to a dynamically created mock.
