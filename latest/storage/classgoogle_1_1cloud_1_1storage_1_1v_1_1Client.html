Read the contents of an object.

Parameters
bucket_namethe name of the bucket that contains the object.
object_namethe name of the object to be read.
parametersa variadic list of optional parameters. Valid types for this operation include IfGenerationMatch/IfGenerationNotMatch, IfMetagenerationMatch/ IfMetagenerationNotMatch, Generation, and UserProject.
Example
void ReadObject(gcs::Client client, int& argc, char* argv[]) {
if (argc < 2) {
throw Usage{
"insert-object <bucket-name> <object-name> <object-contents (string)>"};
}
auto bucket_name = ConsumeArg(argc, argv);
auto object_name = ConsumeArg(argc, argv);
auto stream = client.Read(bucket_name, object_name);
int count = 0;
while (not stream.eof()) {
std::string line;
std::getline(stream, line, '\n');
++count;
}
std::cout << "The object has " << count << " lines" << std::endl;
}
